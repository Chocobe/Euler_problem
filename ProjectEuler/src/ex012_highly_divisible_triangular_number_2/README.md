# Highly divisible triangular number 보완
# (많이 나눠지는 삼각수 보완)

공식을 사용할 경우,

"ex012_highly_divisible_triangular_number" 에서 생겼던 반환시간 문제(5분 넘게 걸림)을 해결할 수 있다.


# 1. 삼각수 구하기
삼각수 구하는 공식은 다음과 같다.
	n이 구하고자 하는 삼각수의 최대 인수값 이라면..
	
	삼각수 = (n + 1) * n / 2
	

# 2. 인수 갯수 구하기
특정값의 인수 갯수를 구하는 공식은 다음과 같다.
	
	a^m * b^n 일 때,
	인수 갯수 == (m + 1)(n + 1)
	
	
# 3. 특정값의 약수(인수)를 구하기 위한 루프 최대값 구하기
 소인수분해를 할 경우, 가장 큰 값(인수)는 "제곱근 + 2"를 넘지 못한다.
 
 때문에 약수(인수)를 구하기 위한 루프의 최대값은 
(제곱근 + 2)보다 작거나 같으면 중복없이 구해진다.
 
뒤에 "+ 2" 에 대해 설명을 하면,
	루프의 시작을 2부터 하기 때문에, "3"에 대한 계산에 예외가 발생한다.
	이유는 루트3 은 "1.7xxx" 이기 때문에 루프가 한번만 돌고 끝나기 때문이다.

	때문에 "+ 2"로 루프값을 보정해 주었다.
