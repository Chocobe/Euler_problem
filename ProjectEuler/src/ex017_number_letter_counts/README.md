# Number letter counts
# (숫자의 문자 총수)

If the numbers 1 to 5 are written out in words:

(만약 1에서 5까지 숫자를 단어로 적게되면 :)

one, two, three, four, five, then there are 3 + 3 + 5 + 4 + 4 = 19 letters used in total.

(one, two, three, four, five 이고, 총 사용 문자는 "3 + 3 + 5 + 4 + 4 = 19" 이다.

If all the numbers from 1 to 1000 (one thousand) inclusive were written out in words,
how many letters would be used?

(만약 1에서 1000(one thousand)을 포함한 모든 숫자들을 단어로 적게된다면, 얼마나 많은 문자들이 사용되겠는가?)

NOTE: Do not count spaces or hyphens. 
For example, 342 (three hundred and forty-two) contains 23 letters and 
115 (one hundred and fifteen) contains 20 letters. 
The use of "and" when writing out numbers is in compliance with British usage.

(NOTE : 공백이나 -는 세지 마시오.
예를 들어, 342 (three hundred and forty-two)는 23개 문자를 포함하고,
115 (one hundred and fifteen)은 20개 문자들을 포함한다.
"숫자를 적을 떄, "and"의 사용은 영국법에 따르는 것이다.)
