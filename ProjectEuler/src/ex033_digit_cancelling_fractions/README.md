# Digit cancelling fractions
# (분수를 바꾸는 숫자)

The fraction 49/98 is a curious fraction, 
as an inexperienced mathematician in attempting to simplify 
it may incorrectly believe that 49/98 = 4/8, which is correct, is obtained by cancelling the 9s.

("49/98" 분수는 이것의 단순화를 시도하는 미숙한 수학자로써 특이한 분수이고,
"49/98 = 4/8"이 부정확하게 믿을지도 모른.
이것은 정확하고, 9를 소거하는것에 의해 포함된다.)

We shall consider fractions like, 30/50 = 3/5, to be trivial examples.

(사소한 예가 되는 "30/50 = 3/5"과 같은 분수를 생각해보자.)

There are exactly four non-trivial examples of this type of fraction, 
less than one in value, and containing two digits in the numerator and denominator.

(1보다 작고, 분자와 분모에 두개의 숫자를 포함하는, 이런 형태의 분수의 특별한 예가 정확히 4개 있다.)

If the product of these four fractions is given in its lowest common terms, 
find the value of the denominator.

(만약 이러한 4개 분수의 결과값이, 이것의 가장 낮은 공통항이 주어진다면, 이 값의 분모를 찾아라.)

# 문제점_1
문제의 해석이 어려웠다.

자력 해석으로 문제를 이해할 수 없었기에, kor euler에서 문제를 이해했다.

문제의 요구사항에 대해서는 크게 어려움은 없었다.

다만, 형변환 (float)를 사용했다는 점이 맘에 안든다.
