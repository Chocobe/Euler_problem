# Circular primes
# (회전 소수)

The number, 197, is called a circular prime 
because all rotations of the digits: 197, 971, and 719, are themselves prime.

("197"이라는 숫자는 "197, 971, 719"인 숫자의 모든 회전이 그들 스스로 소수이기 때문에 회전 소수라고 불린다.)

There are thirteen such primes below 100: 2, 3, 5, 7, 11, 13, 17, 31, 37, 71, 73, 79, and 97.

(100이내의 이러한 소수는 13개 있다.)

How many circular primes are there below one million?

(100만 이내에 회전 소수는 몇개 있는가?)
