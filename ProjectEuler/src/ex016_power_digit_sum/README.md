# Power digit sum
# (제곱 수 합)

2^15 = 32768 and the sum of its digits is 3 + 2 + 7 + 6 + 8 = 26.
(2^15 = 32768에서 이것의 숫자들의 합은 "3 + 2 + 7 + 6 + 8 = 26" 이다.)

What is the sum of the digits of the number 2^1000?
(2^1000의 숫자들의 합은 얼마인가?)

# 문제점_1
피보나치 문제에서도 걸렸던 자료형의 범위가 문제였다.
4자리씩 나눈 값을 배열단위로 계산하고자 했지만...
기하급수적으로 늘어나는 다항식을 메서드로 만들지 못했다.

그래서 기본 자료형보다 큰 것을 찾은 결과,
BigInteger 라는 클래스가 존재했다.

BigInteger는 사칙연산으로 +-*/ 를 사용하지 않고, 해당 메서드를 호출해서 사용해야 한다.

값의 범위는 "-2 ^ (2 ^ 31) ~ 2 ^ ((2 ^ 31) - 1)"
엄청나다...

BigInteger의 범위를 넘는 문제가 나오지 않는 한,
이후로는 자료형의 크기때문에 헤매는 일은 사라질 것이라 기대한다.
