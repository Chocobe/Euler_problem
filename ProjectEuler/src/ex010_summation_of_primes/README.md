# Summation of primes
# (소수들의 총합)

The sum of the primes below 10 is 2 + 3 + 5 + 7 = 17.
(10 미만의 소수들의 합은 "2 + 3 + 5 + 7 = 17" 이다.)

Find the sum of all the primes below two million.
(2백만 미만의 모든 소수들의 합을 구하시오.)

# 소수 판별
소수를 판별하기 위해, 나눌 값의 범위를 "sqrt타겟값" 이하로 주었다.
(이 값 이상의 값은 지금껏 연산했던 값과 동일한 부분이기 때문이다.)

이렇게 해서 나온 "약수의 갯수"가 2개 이상이라면, 소수가 아니다.
